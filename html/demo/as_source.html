展示标签名

可以使用 ::before 和 ::after 在标签前后创建 伪元素，并设置伪元素的内容为标签名。

如果能一次性为所有的元素都指定 before/after 内容就好了：

*::before{
        content: '<' name() '>'
        }
        *::after{
        content: '</' name() '>'
        }

但是伪元素(还)不支持name()这种用法。原文是枚举出网页中用到的标签，然后手写规则。为了通用性，我们也可以用 js 来动态控制（虽然这样就不是纯 CSS 方法了）。
